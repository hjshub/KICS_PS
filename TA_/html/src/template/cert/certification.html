행정전자문서의 안정적 유통을 위한
정부차원의 보안 인증시스템!

인증서 가져오기 안내

 

  1. PC에서 KICS 인증관리센터 메뉴(메뉴 경로 설명)에 접속해 주세요.
  2. PC화면의 [스마트폰으로 인증서 내보내기] 버튼을 선택해 주세요.
  3. 스마트폰으로 복사할 인증서를 선택하신 후 인증서 비밀번호를 입력해 주세요.
  4. [인증서 가져오기] 버튼 선시 스마트폰에 생성된 승인번호 12자리를 확인합니다.
  5. PC화면에 스마트폰에 생성된 승인번호 12자리를 입력 후 [내보내기] 버튼을 선택해 주세요.
  6. 스마트폰에서 해당 인증서를 저장해 주세요.
  7. 스마트폰에 저장된 인증서를 확인 해 주세요.
